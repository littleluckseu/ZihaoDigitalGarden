## 结构体、枚举类型标注展开目录

```text
/**
 * @brief 简要说明
 *
 * @details 详细说明
 */
typedef enum BoxEnum_enum {
  BOXENUM_FIRST,  /**< Some documentation for first. */
  BOXENUM_SECOND, /**< Some documentation for second. */
  BOXENUM_ETC     /**< Etc. */
} BoxEnum;
```

## 函数头注释展开目录

```text
/** 
 * @brief 函数简要说明, 可以换行
 *
 * @details 对函数的详细说明和解释，可以换行
 *
 * @note 注意事项内容
 *
 * @param index 说明文字
 * @param cent 说明文字
 *
 * @return 描述函数返回的内容
 * @retval 1 测试成功
 * @retval -1 测试失败
 */
bool Test(int index, char *cent);
```

## 变量注释展开目录

```text
int element = 0; /**< 变量简单说明 */
```

```text
/**
 * @brief 变量简要说明
 * 
 * @details 变量详细说明
 */
unsigned int variable = 0;
```
